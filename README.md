# 🎯 Sistem Absensi QRCode - Laravel 12

Sistem absensi berbasis QRCode dengan Laravel 12, tanpa Vite/NPM, siap deploy ke hosting.

## 🚀 Fitur Utama
- ✅ Multi user (Admin & Karyawan)
- ✅ Generate QR Code dengan timeout
- ✅ Scan QR Code via web browser camera
- ✅ Anti-cheat system (rate limiting, one-time use)
- ✅ Laporan lengkap dengan export CSV
- ✅ No page refresh (AJAX + JavaScript)
- ✅ Responsive design dengan SB Admin 2
- ✅ Security lengkap

## 📋 Requirements
- PHP 8.2+
- MySQL 5.7+
- Composer
- Web server (Apache/Nginx)
